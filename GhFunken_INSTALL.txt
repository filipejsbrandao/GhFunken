IMPORTANT NOTE: This version of Funken is only compatible with Rhino8. If you are using or need compatibility with a previous version of Rhino
please check the original Funken project available on github.

-----STEP#01 - Funken Library -----
1 - Unzip this folder
2 - Copy the "GhFunken" folder to Grasshopper Components folder (File -> Special folders -> Components)
-----

-----STEP#02 - Funken User Objects -----
1 - Copy the content of the userObjects folder into Grasshopper user objects folder (File -> Special folders -> User Objects)
2 - Reload Grasshopper and the Funken tab should appear.
-----
